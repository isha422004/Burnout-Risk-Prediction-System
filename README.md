# Burnout-Risk-Prediction-System
This project predicts burnout risk based on lifestyle indicators such as sleep hours, stress level, screen time, workload, and career clarity.

Technologies Used:
- Python
- Machine Learning (Decision Tree)
- Streamlit
- Numpy
- Pandas
- Matplotlib
- Scikit-learn

Features:
- Burnout Risk Prediction
- Early Intervention Advisory
- Personalized Recovery Strategies
- Explainable AI-based reasoning
